I design and build **Genesis Agent** — a self-aware, self-modifying AI agent with a 12-phase boot system, hexagonal architecture, and a learning flywheel that gets faster with every task. It doesn't just use LLMs — it decides *when not to*, bypassing the model entirely for problems it has already solved.

### Key systems

| **SymbolicResolver** | Checks LessonsStore before calling the LLM — known solutions execute instantly |
| **EarnedAutonomy** | Wilson-score trust per action type, auto-promotes at 85% confidence |
| **DaemonController** | Unix Socket / Named Pipe RPC — control Genesis from any terminal |

| **LessonsStore** | Cross-project persistent learning (`~/.genesis-lessons/`) |

| **PreservationInvariants** | Hash-locked semantic safety rules |
| **SelfModificationPipeline** | Plan → test → snapshot → apply, with consciousness gate |

### Evolution

Genesis v7 is the result of deliberate subtraction. The Consciousness Layer (14 modules, 6198 LOC) was benchmarked at 0% impact and replaced by a 112 LOC interface. The codebase shrank by 6k lines and got stronger — 100% event schema coverage, 100% file test coverage, every silent catch eliminated.

```
v5.9 → v6.0: Self-modification pipeline, organism layer, 13-phase boot
v6.0 → v6.1: Learning Flywheel, observability, gate statistics
v6.1 → v7.0: Consciousness removed, AwarenessPort, AgentLoop split, Colony IPC
v7.0 → v7.0.1: 100% schema coverage, DaemonController, dead code purge
```
